# Digital Thermometer using Arduino

## Overview

This project is a digital thermometer designed using Arduino and a temperature sensor in Tinkercad. The system measures ambient temperature and displays the temperature readings in real time.

## Features

* Real-time temperature measurement
* Simple and easy-to-understand design
* Arduino-based implementation
* Simulated and tested using Tinkercad Circuits

## Components Used

* Arduino Uno
* Temperature Sensor (LM35/TMP36)
* Breadboard
* Connecting Wires
* LCD Display

## Working Principle

The temperature sensor continuously measures the surrounding temperature and sends an analog signal to the Arduino. The Arduino converts this signal into temperature values and displays the readings through the  LCD display.

## Development Platform

This project was designed, programmed, and tested using Tinkercad Circuits Simulation.

## Learning Outcomes

* Arduino programming fundamentals
* Analog sensor interfacing
* Temperature measurement concepts
* Circuit simulation using Tinkercad

## Future Improvements
